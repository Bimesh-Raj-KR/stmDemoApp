# stmDemoApp

A C project to blink LEDs and print data using UART and GPIO pins. 

## Features

Organized source code in multiple directories
Clean, portable build system

## Prerequisites

**STM32CubeIDE** ≥ 1.18.1
**EGit** ≥ 6.7.0

## Building with Make

### 1. Build All

$ make -j8 all 

Compiles the Project

### 2. Clean Build Artifacts

$ make clean

Removes Intermediate build files and Binaries
